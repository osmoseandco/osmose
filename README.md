# osmose
Plateforme intelligente qui centralise tous les événements autour de soi en temps réel (gratuits, payants, bien-être, concerts, influenceurs, tourisme, expériences locales).  L’objectif est de créer une expérience fluide, addictive et inspirante, entre :  Airbnb Experiences Eventbrite Instagram (exploration) Google Maps (géolocalisation)
